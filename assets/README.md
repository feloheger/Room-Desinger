# Assets

Lege hier folgende Dateien ab (Platzhalter-Dateinamen aus app.json):

- icon.png            (1024x1024, App-Icon)
- splash.png          (Splash-Screen-Bild, transparent/weißer Hintergrund)
- adaptive-icon.png   (1024x1024, Android Adaptive Icon Vordergrund)
- favicon.png         (48x48, für Web)

Für den "Clean White" Look empfiehlt sich ein simples, monochromes
Icon (z.B. ein Haus- oder Sofa-Symbol in #0A0A0A auf transparentem
oder weißem Grund).
